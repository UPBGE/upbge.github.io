gpu_extras.presets.draw_circle_2d(position, color, radius, *, segments=None)ÔÉÅ

Draw a circle.

Parameters:

  • position (Sequence[float]) ‚Äì 2D position where the circle will be drawn.

  • color (Sequence[float]) ‚Äì Color of the circle (RGBA). To use transparency blend must be set to ALPHA, see: gpu.state.blend_set().

  • radius (float) ‚Äì Radius of the circle.

  • segments (int | None) ‚Äì How many segments will be used to draw the circle. Higher values give better results but the drawing will take longer. If None or not specified, an automatic value will be calculated.
